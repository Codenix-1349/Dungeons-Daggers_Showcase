## 📌 Overview

Dungeons & Daggers is a full-stack web application for AI-supported solo adventures inspired by the D&D SRD 5.2.1.

Players can create characters, import adventures, start persistent sessions, and play through browser-based RPG scenarios with AI-supported narration and structured gameplay logic.

---

## 🛠 Tech Stack

| Area | Technologies |
|------|--------------|
| Frontend | React, Vite |
| Backend | Node.js, Express |
| Database | PostgreSQL |
| AI Integration | OpenRouter |
| File Processing | PDF.js, TXT import |
| Authentication | JWT, email verification, password reset |
| Testing | Vitest |

---

## ✨ Core Features

- Character creation and management
- AI-guided solo adventure gameplay
- Persistent game sessions
- PDF and TXT adventure import
- Combat, checks, progression, and loot handling
- User registration and login
- Email verification and password reset
- Backend-based session and account handling
- Secure API key handling for AI usage
